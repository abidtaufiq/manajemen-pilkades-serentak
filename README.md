# APLIKASI PILKADES SERENTAK

Dibuat menggunakan:
- Codeigniter
- Bootstrap 3.7
- AdminLTE2
- PHPExcel/PHPOffice
- mPDF
- jQuery/AJAX
- ChartJS
- DataTable
- Sweet Alert
- dll


Fitur : 
- Multi level user
- CRUD dengan Ajax
- Datatable
- Export ke format Excel
- Cetak laporan ke format Excel dan PDF
- Pie/Bar Chart
- Time Schedule (tahapan pengolahan data terjadwal)
